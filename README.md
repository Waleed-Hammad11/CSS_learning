# 🚀 Frontend Mini-Projects Collection

Welcome to my HTML and CSS mini-projects repository\! This collection showcases a variety of user interfaces, interactive components, and fully responsive web layouts built from scratch.

The primary goal of this repository is to demonstrate a solid understanding of fundamental web development concepts, including modern CSS techniques, responsive design principles, and UI/UX best practices.

-----

## 📂 Project Categories

To make navigation easier, I have categorized the 16 projects based on their primary focus and structural complexity:

### 🌐 Complete Websites & Landing Pages

These projects focus on overall page composition, visual hierarchy, and brand identity.

  * **Avatar Website:** A thematic web page inspired by the Avatar movie universe.
  * **Pepsi Website:** A promotional product landing page focusing on brand colors and layout.
  * **Simple Portfolio Website:** A clean, personal portfolio template to showcase skills and previous work.
  * **Testimonials Landing Page:** A grid-based layout designed to display user reviews attractively.
  * **Glassmorphism Website:** A modern UI implementation utilizing CSS `backdrop-filter` to create a frosted glass effect.

### 📐 Layouts & Structural Concepts

These projects demonstrate proficiency in positioning elements and creating robust page architectures.

  * **Holy Grail Mockup (Grid2):** A classic web page layout (Header, Sidebar, Main Content, Footer) built entirely with modern CSS Grid.
  * **Odin Project Dashboard:** A complex, responsive dashboard UI implementing advanced CSS Grid and Flexbox techniques.
  * **Side Navigation Bar:** A collapsible, interactive sidebar for efficient user navigation.
  * **Responsive Website Footer:** A multi-column footer that gracefully adapts to mobile and desktop screens.

### 🔐 Interactive Forms & Inputs

Focusing on form validation, user input, and aesthetic styling of interactive elements.

  * **Odin Sign Up:** A comprehensive registration form featuring client-side CSS validation and interactive states.
  * **Responsive Login Form:** A fluid authentication interface that looks great on any device.
  * **Login Form with Icon:** A visually enhanced login card integrating vector icons for better user experience.
  * **Responsive Search Box:** An expandable search input field that responds smoothly to user interaction.

### 🎨 Creative UI Components

Bite-sized projects focusing on aesthetics, shadows, and detailed CSS styling.

  * **Black Master Card:** A realistic, dark-themed credit card UI featuring gradients and subtle reflections.
  * **Profile UI Card:** An elegant user profile component ideal for social media or dashboard environments.
  * **Coder Girl:** A creatively styled UI component reflecting a developer-themed identity.

-----

## 🛠️ Technologies & Tools Used

  * **HTML5:** Semantic markup for accessibility and SEO optimization.
  * **CSS3:**
      * **Flexbox & Grid:** For one-dimensional and two-dimensional layouts.
      * **Media Queries:** Ensuring a mobile-first, fully responsive experience.
      * **CSS Variables (Custom Properties):** For maintainable and scalable theming.
      * **Transitions & Animations:** Adding smooth hover effects and dynamic interactions.

## 💻 How to Run Locally

If you wish to explore the code on your local machine, follow these steps:

1.  Clone the repository to your local environment:

    ```bash
    git clone https://github.com/YourUsername/YourRepoName.git
    ```

2.  Navigate into the specific project folder you want to view:

    ```bash
    cd YourRepoName
    ```

3.  Open the `index.html` file of the chosen project directly in your preferred web browser.
